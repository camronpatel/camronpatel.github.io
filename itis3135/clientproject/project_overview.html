Project Overview

Application and Purpose

The JavaScript Basics is a educational web page that is designed to teach the basics of JavaScript. The purpose of this webpage is to provide a fun and interactive place for people to start learning JavaScript. Topics can include include variables, data types, operators, functions, and events through interactive lessons, quizzes, and examples.

The purpose of this webpage is to provide a user-friendly platform for people to learn JavaScript.

Wireframe

SiteMap

Page Design

  • Home (index.html): Introduction and navigation. Audience: Everyone. No inputs. Uses jQuery and AJAX tip loader.
  • Lessons: Educational articles and JS syntax examples. Audience: Learners. Static content.
  • Quiz: Multiple-choice quiz form. Audience: Students. Inputs with Quiz form. Will Contain buttons, data validation for the quiz aswell.
  • Practice: Code input and output display using eval(). Audience: All users. Input box with JS execution.
  • About: Project background and purpose. Audience: General.

Dynamic Functionality

  • Accordion UI: #accordion element on homepage using jQuery UI for structured intro display.
  • AJAX Tip Loader: Button fetches a random helful Javascript tip from a JSON file
  • Practice Page: Code typed in a textarea is executed with output shown in a preformatted block.
  • Quiz: Quiz form captures and evaluates user answers using JavaScript DOM.
