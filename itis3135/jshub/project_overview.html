Application and Purpose

The JavaScript Basics is a educational web page that is designed to teach the basics of JavaScript. The purpose of this webpage is to provide a fun and interactive place for people to start learning JavaScript. Topics can include include variables, data types, operators, functions, and events through interactive lessons, quizzes, and examples.

The purpose of this webpage is to provide a user-friendly platform for people to learn JavaScript.

Wireframe

SiteMap

Page Design

  • Home (index.html): Introduction and navigation. Audience: Everyone. No inputs. Uses jQuery and AJAX tip loader. The page will contain a button for the random tip. User will need to click through headings to show the information.
  • Lessons: Educational content and JS syntax examples. Audience: Learners. The page will contains simple examples of core JavaScript concepts such as: Variables, Data Types, Functions, Conditionals, Loops, and Events.
  • Quiz: Multiple-choice quiz form. Audience: Students. Inputs with Quiz form. Will contain buttons for each quiz question and a submit, data validation for the quiz. The submit button will yield a score based on the questions answered in the quiz.
  • FAQ:Provides answers to Frequenently Asked Questions for JavaScript beginners ("When to use let vs const"). Audience: Learners. Uses accordion UI and jQuery UI for a dropdown menu of questions and answers.
  • About: Project background and purpose. Audience: Everyone.

Dynamic Functionality

  • Accordion UI: #accordion element on homepage using jQuery UI for structured intro display. Also used in FAQ that provides users with answers to Frequenently Asked Questions.
  • AJAX Tip Loader: Button fetches a random helful Javascript tip from a JSON file.
  • Quiz: Quiz form captures and evaluates user answers using JavaScript DOM.
