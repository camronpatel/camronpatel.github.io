JavaScript Lessons

Variables

Variables store data values. JavaScript uses var, let, and const to declare variables. let is blockâ€‘scoped and is preferred over var. const is used for variables that do not change.

let age = 25;
const name = "Aaria";
var score = 90;

Data Types

JavaScript has several data types including strings, numbers, booleans, arrays, and objects.

let text = "Hello";        // String
let number = 5;              // Number
let isHappy = true;          // Boolean
let list = [1, 2, 3];        // Array
let person = {name: "Cam", age: 20}; // Object

Functions

Functions group code into reusable blocks. They are defined using the function keyword or arrow syntax.

function greet(name) {
  return "Hello, " + name + "!";
}

const greetArrow = (name) => "Hi, " + name + "!";

Conditionals

Conditional statements control the flow of your program using if, else if, and else.

let score = 85;
if (score >= 90) {
  console.log("A");
} else if (score >= 80) {
  console.log("B");
} else {
  console.log("C or below");
}

Loops

Loops execute code multiple times. Common types include for, while, and forEach.

for (let i = 0; i < 3; i++) {
  console.log(i);
}

let i = 0;
while (i < 3) {
  console.log(i);
  i++;
}

Events

Events let you respond to user actions like clicks or keypresses.

document.getElementById("btn").addEventListener("click", function() {
  alert("Button clicked!");
});
