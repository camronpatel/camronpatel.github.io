Frequently Asked Questions

What is JavaScript used for?

JavaScript is a programming language that is used in web development. It is used to create interactive web pages, build server-side applications, and more.

What's the difference between let and const?

let assigns a variable that can be reassigned, while const is a variable that connot be reassigned

How do I debug my JavaScript code?

You can use console.log() to print values, at certain points in your code for helpful debugging.

What is an event listener?

An event listener waits for a specific event (like a click) and runs a function when that event occurs. Example:

element.addEventListener('click', () => {
  console.log('Clicked!');
});
